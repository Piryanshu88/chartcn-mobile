## 📝 About This Project

chartcn-mobile uses open, versioned ChartSpec files to ensure your charts stay consistent across mobile platforms.

It supports modern mobile frameworks like Jetpack Compose for Android and SwiftUI for iOS.

The system handles a range of chart types and simple JSON data formats.

This makes it easy to work with data visually on your phone without needing to learn coding or complex software.

## 📂 File Types and Formats

The main files used in chartcn-mobile are:

- **ChartSpec files**: These are JSON files that describe the charts.
- **Data files**: These contain the raw data your charts represent.
- **Exported charts**: You can save charts as images or PDFs for sharing.

You can open, edit, or create these files using simple text editors or with chartcn-mobile's built-in tools.
